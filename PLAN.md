# Brick Breaker 開発計画（現行）

最終更新: 2026-02-14

## 現状（実装済み）
- Bun + Vite + TypeScript + Canvas 構成
- 12ステージキャンペーン
- 6種アイテム（`paddle_plus`, `slow_ball`, `multiball`, `shield`, `pierce`, `bomb`）
- 落球時の全アイテム解除
- ステージクリア時のアクティブ効果持ち越し（落下中アイテムは破棄）
- 物理・描画・アイテム処理のシステム分割

## 残タスク（未完了のみ）
1. コンボ倍率システム
   - 連続破壊の時間窓で倍率を上げる
   - 落球時に倍率リセット
2. ステージ評価（星1〜3）
   - クリアタイム / 被弾回数 / 残機で評価
   - ステージクリア画面に表示
3. エリートブロック導入
   - 耐久2または耐性ブロックを後半ステージへ限定配置
4. アクセシビリティ強化
   - 色覚多様性向けテーマ
   - reduced-motion時の演出差分をUIで明示
5. UIテーマ拡張
   - ステージ帯ごとの配色プリセット
   - HUDの視認性を高DPI環境で再調整

## 実施順（推奨）
1. コンボ倍率
2. ステージ評価
3. エリートブロック
4. アクセシビリティ強化
