# Brick Breaker 開発計画（現行）

最終更新: 2026-02-14

## 現状（実装済み）
- Bun + Vite + TypeScript + Canvas 構成
- 12ステージキャンペーン
- 6種アイテム（`paddle_plus`, `slow_ball`, `multiball`, `shield`, `pierce`, `bomb`）
- 落球時の全アイテム解除
- ステージクリア時のアクティブ効果持ち越し（落下中アイテムは破棄）
- コンボ倍率システム（1.8秒窓、`x1.0 -> x3.0`）
- ステージ評価（星1〜3）とクリアリザルト表示
- エリートブロック（`durable`/`armored`）を9〜12面へ導入
- 1〜4 / 5〜8 / 9〜12 面のテーマ帯切替（背景・ブロックトーン・HUDアクセント）
- 主要回帰テスト拡張（コンボ・評価・エリート・ステージ遷移）
- 物理・描画・アイテム処理のシステム分割
