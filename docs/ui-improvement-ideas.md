# Brick Breaker UI改善アイデア（残タスク）

最終更新: 2026-02-14

## 現状（実装済み）
- グラス調HUDとシーンオーバーレイ
- ヒット時演出（パーティクル、リング、シェイク、フラッシュ）
- スコアポップ、進捗バー、ボール位置インジケータ
- 高DPI描画対応
- reduced-motion対応

## 残タスク（未完了のみ）
1. コンボ表示UI
   - 連続破壊倍率の可視化
   - 短時間で読める表示位置と寿命調整
2. ステージ評価UI
   - クリア時に星評価と内訳（時間/被弾/残機）を表示
   - `stageclear` / `clear` 画面の情報量を整理
3. 色覚多様性テーマ
   - ブロック色・アイテム色の識別性改善
   - 通常テーマとの切替導線追加
4. 後半ステージ向けテーマ差分
   - 9〜12面で背景/エフェクト色を段階変更
   - 視認性を維持したまま達成感を強化
5. reduced-motion可視化
   - 設定反映中であることをHUDで明示
   - 演出が抑制される理由を簡潔に表示

## 導入順（推奨）
1. コンボ表示UI

3. 色覚多様性テーマ
4. 後半ステージ向けテーマ差分
