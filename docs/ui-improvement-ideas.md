# Brick Breaker UI改善アイデア（残タスク）

最終更新: 2026-02-14

## 現状（実装済み）
- グラス調HUDとシーンオーバーレイ
- ヒット時演出（パーティクル、リング、シェイク、フラッシュ）
- スコアポップ、進捗バー、ボール位置インジケータ
- 高DPI描画対応
- reduced-motion対応
- コンボ表示（HUD `COMBO xN`）
- ステージリザルト表示（星評価、時間、被弾、残機）
- 後半ステージのテーマ帯切替（背景/ブロック/HUDアクセント）
- エリートブロック表示（耐久数字、特殊枠・刻印）

## 残タスク（未完了のみ）
1. 色覚多様性テーマ
   - ブロック色・アイテム色の識別性改善
   - 通常テーマとの切替導線追加
2. reduced-motion可視化
   - 設定反映中であることをHUDで明示
   - 演出が抑制される理由を簡潔に表示
3. ステージリザルトの視認性最適化
   - スマホ横幅時の改行崩れ対策
   - 星評価と内訳の強弱（階層）整理

## 導入順（推奨）
1. 色覚多様性テーマ
2. reduced-motion可視化
3. ステージリザルトの視認性最適化
